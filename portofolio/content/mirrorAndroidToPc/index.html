Tutorial mirror layar android ke ubuntu/xubuntu

Sebelum kita sampai pada bagian yang menyenangkan dari mirroring Android di Linux, Anda harus mengaktifkan debugging USB di ponsel Anda .

1. buka setting => system => tentang ponsel => klik build number 7x sampai keluar tulisan sekarang kamu adalah seorang developer.

2. sekarang aktifkan usb debugging . sebelum tentang ponsel => opsi developer => nyalakan usb debugging . ada tulisan muncul klik ok.. lihat gambar!

Saatnya keubuntu / xubuntunya.

install scrcpy dulu lewat terminal .. copy paste kode dibawah diterminal.
dan enter. lihat gambar!

  • Hubungkan hp android ke laptop dengan kabel usb

  • Buka terminal dan ketik scrcpy terus enter

  • Diponsel akan ada pop up persetujuan ceklist kotak selalu izinkan dari komputer ini terus klik ok... lihat gambar!

  • Hp kamu sekarang sudah kemirror kelaptop. jika belum kamu tinggal tulis scrcpy diterminal dan enter... lihat gambar!
